# VoteEverything

**Moderní hlasovací a komunitní aplikace**

Hlasuj, vytvářej ankety, chatuj přes QR kód a sbírej body v bojovkách.

## Funkce

- Procházení a hlasování v anketách
- Vytváření vlastních anket
- Soukromé chaty přes QR párování (**Minichat**)
- Chat přímo se správcem
- BOJOVKA QR – sbírání bodů a odemykání funkcí
- Profil s avatarem a unikátním QR kódem
- Podpora anonymních, PRO a ULTIMATE účtů

## Technologie

- Flutter + Dart
- Firebase (Auth, Firestore, Storage)
- QR kód párování

# VoteEverything

**Modern voting and community app**

Vote, create polls, chat via QR code and collect points in battlegrounds.

## Features

- Browse and vote in polls
- Create your own polls
- Private chats via QR pairing (**Minichat**)
- Chat directly with the administrator
- QR BATTLE – collect points and unlock features
- Profile with avatar and unique QR code
- Support for anonymous, PRO and ULTIMATE accounts

## Technologies

- Flutter + Dart
- Firebase (Auth, Firestore, Storage)
- QR code pairing
